# Approaches for data analysis in Python

## Finding if two lists have a shared element

- There are at least four ways to approach this problem. With great details mentioned in the [Stackoverflow question](https://stackoverflow.com/questions/3170055/test-if-lists-share-any-items-in-python)

- Using `any()` provides an easy solution and can be impemented by the following `any(i in neglect_strings for i in quad.split('_'))`. In short, the built-in function `any()` behaves like a logical *OR* operator (`|`,`and()` in MATLAB), and `all()` behaves like a logical *AND* operator (`&`, `and()` in MATLAB)
